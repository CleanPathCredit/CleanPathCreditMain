# Spanish Video Scripts — 10 Starter Scripts

First tranche of the 30-video Spanish content plan (Track A4/B2 hybrid). Designed to drive traffic to `/es-comprador` and the Calendly call. Same Latino working-class first-time buyer avatar as the consumer one-pager.

**Sequencing logic:** ship these 10 first, batched into two 2-hour filming sessions, posted across TikTok / IG Reels / FB Reels / YouTube Shorts over 2 weeks. Track which 3 themes hit (saves, shares, comments — not just views), then write the next 20 scripts in those themes. Spanish-language credit content has dramatically lower CPM than English on Meta and TikTok; front-load Spanish over English.

---

## Pre-production

### Format spec
- **Length:** 30–60 seconds. Hook in the first 3 seconds. CTA in the last 5.
- **Aspect:** 9:16 vertical. Same master file works for TikTok / IG Reels / FB Reels / YouTube Shorts.
- **Captions:** Spanish captions burned in (don't trust auto-captions for Spanish — they hallucinate). 2/3 of viewers watch sound-off.
- **Audio:** lavalier mic mandatory. AirPods or phone mic kills Spanish-content trust instantly.
- **Lighting:** natural window light or single softbox. Avoid harsh fluorescent.
- **Wardrobe:** business-casual but warm (button-up, no suit, no tie). Looking too "corporate" reads as the radio scammers' lawyer.
- **Background:** home, kitchen, neighborhood walk, taqueria, parking lot. NOT a sterile office. Avatar trusts in-context, not in-conference-room.

### Filming kit (one-time setup, ~$200)
- Phone tripod with phone mount
- Ring light or single softbox
- Lavalier mic (Rode Wireless ME or Boya BY-M1 wired)
- Second phone for B-roll capture
- Optional: small whiteboard for hand-drawn explainers (high-converting on TikTok credit content)

### Two batched filming sessions
- **Session 1 (2 hours, talking-to-camera at home):** Scripts 1, 4, 5, 6, 10. Same outfit, same location, change angle/background slightly between scripts. Goal: 5 finished talking-head clips.
- **Session 2 (2 hours, walking-and-talking + B-roll):** Scripts 2, 3, 7, 8, 9. Varied locations — home, neighborhood walk, taqueria, parking lot. Heavy B-roll for cutaways.
- Re-shoot any take where you stumble. The avatar can hear performative Spanish from a mile away — if it doesn't sound like you talking to your prima at a carne asada, re-shoot.

### Universal CTA template (last 3–5 sec, every video)

> "Si tienes preguntas, link en mi bio para una llamada gratis en español. Quince minutos. Sin compromiso."

Link in bio points at `cleanpathcredit.com/es-comprador` (or shortlink). Set up a Spanish-funnel UTM: `?utm_source=tiktok&utm_medium=organic&utm_campaign=es_video_<script_number>` so GA's `spanish_funnel_view` event attributes per video.

## Compliance — absolute hard rules

**Things that MUST NOT appear in any video, ever, even off-script:**

- "Te garantizo que [outcome]" — CROA §404(a)(3) prohibits any guarantee
- "Yo te subo el score X puntos en X días" — specific timeline + score = double violation
- "Te apruebo la hipoteca" — you don't approve, the LO does
- "Borramos tu mal crédito" — the radio scammer phrase
- "Removemos cuentas en X días" — specific timeline
- Comparisons to specific competitors by name (FTC defamation/false-comparison risk)
- Specific dollar amounts of "interest savings" or "down payment money saved"
- Claims about a real client outcome unless you have written substantiation on file (FTC Endorsement Guides updated 2023, §465.2 — $51,744 per violation per view)

**On-screen text disclaimer required on any video that mentions score improvement or program timelines:**

> *"Resultados varían. No garantizamos cambios específicos. Consulta nuestros términos."*

Display 4-6 seconds, white text on a dark band, bottom of screen. Don't bury it in the last frame — must be readable.

**Texas CSO registration:** until your registration number is approved and posted, do NOT use the words "reparación de crédito" / "credit repair" in the videos as a service you currently sell. Frame as "educación financiera" / "preparación hipotecaria" / "información sobre crédito." Once registered, switch the framing.

## Post-production

### Editing
- Cut tight: trim every "um," every dead second. Spanish-content average attention span on TikTok is 12 sec. Get to the value fast.
- Add text overlays for the 2-3 key phrases per video. Reinforces sound-off viewing.
- Burn the disclaimer in as a non-removable text band, not a separate caption track.
- End card: 2 seconds, `cleanpathcredit.com/es-comprador` + Clean Path logo. Same on every video for brand recall.
- Music: original audio for 9/10 talking-head scripts. Trending TikTok commercial-license audio for B-roll only (Script 2, 7).

### Posting cadence (2 weeks, 10 videos)

| Day | Videos posted | Platform priority |
| --- | ------------- | ----------------- |
| Mon Wk1 | Script 1 (FICO 10T explainer) | TikTok 8am, IG Reels 10am, FB Reels 12pm |
| Wed Wk1 | Script 5 (cash income docs)   | Same cadence |
| Fri Wk1 | Script 8 (no advance fees)    | Same cadence |
| Sat Wk1 | Script 4 (ITIN mortgages)     | Saturday boosts in Latino markets |
| Mon Wk2 | Script 2 (3-piece change)     | Same cadence |
| Wed Wk2 | Script 9 (3-day cancel right) | Same cadence |
| Fri Wk2 | Script 6 (mixed family)       | Same cadence |
| Sat Wk2 | Script 7 (no credit cards)    | Saturday |
| Mon Wk3 | Script 3 (score went up)      | Wraps tranche |
| Wed Wk3 | Script 10 (one specialist)    | Wraps tranche |

Don't post all 10 in week 1 — the algorithm penalizes burst posting on a new account. 4-5 per week is the sweet spot.

### Tracking
- Each video's bio link uses UTM `?utm_source=tiktok&utm_medium=organic&utm_campaign=es_video_<NN>`. Same UTM convention across IG Reels (`utm_source=ig`) and FB Reels (`utm_source=fb`).
- GA event `spanish_funnel_view` already firing on `/es-comprador` mount — segment by `utm_source` and `utm_campaign` to see which videos drive funnel views.
- After 14 days, score each video on: views, watch-time %, saves, comments, profile visits, link clicks (TikTok Pro Analytics + IG Insights). Saves and comments matter MORE than views — they predict re-watch and share velocity.
- The 3 scripts with the highest combined save+comment+link-click rate become the themes for the next 20 scripts.

### Composio automation (when wired locally)

Once the local Composio MCP is connected, this flow can run nightly:

1. Pull TikTok / IG / FB analytics for each posted video
2. Compute the score: `(saves + comments + link_clicks) / views`
3. Log to a Google Sheet (Clean Path > Spanish Video Performance)
4. Slack daily digest: top 3 videos, bottom 3 videos, and any video that crossed 10K views overnight

When ready to wire, ask local Claude Code to draft the Composio runbook — same pattern as the testimonial-collection automation discussed earlier.
